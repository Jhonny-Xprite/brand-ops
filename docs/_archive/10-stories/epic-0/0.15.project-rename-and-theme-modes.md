## Story
**Como** operador do sistema,  
**Eu quero** renomear o projeto nas configuracoes e alternar entre dark mode e light mode,  
**Para que** o contexto do projeto fique editavel e a experiencia visual se adapte ao meu uso real.

## Acceptance Criteria
1. A tela `/projeto/[id]/config` permite editar o nome do projeto
2. O endpoint canonico de configuracao persiste o nome novo no `Project`
3. O estado global de projetos reflete a renomeacao sem depender de recarga manual
4. O aplicativo suporta dark mode e light mode com persistencia local
5. O toggle de tema fica acessivel nas superficies principais de navegacao
6. `lint`, `typecheck`, `build` e `test` passam ao final

## Tasks

- [x] Atualizar o slice de projetos para refletir renomeacao em runtime
- [x] Implementar provider e toggle de tema com persistencia local
- [x] Registrar evidencias, file list e resultado de QA
