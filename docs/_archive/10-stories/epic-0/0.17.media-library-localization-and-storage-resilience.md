## Story
**Como** operador do workspace de projeto,  
**Eu quero** que a `MEDIA LIBRARY` esteja localizada em PT-BR e que o upload use um storage local resiliente,  
**Para que** a experiencia nao misture idiomas e nao dependa de um drive externo fixo para funcionar.

## Acceptance Criteria
1. A `MEDIA LIBRARY` deixa de exibir strings visiveis em ingles nas superficies principais de busca, filtros, acoes, cards, estado vazio e painel
2. O upload deixa de depender exclusivamente do path fixo `E:\\BRAND-OPS-STORAGE`
3. O sistema passa a usar um storage root configuravel via ambiente com fallback seguro para `data/storage`
4. Os erros de filesystem expostos ao usuario ficam em PT-BR
5. `lint`, `typecheck`, `build` e `test` passam ao final

## Tasks
- [x] Localizar as strings visiveis restantes da `MEDIA LIBRARY`
- [x] Introduzir um helper compartilhado para resolver o storage root com fallback local
- [x] Remover hardcodes de `E:\\BRAND-OPS-STORAGE` dos fluxos de upload/versionamento
- [x] Traduzir mensagens de erro de filesystem expostas ao operador
- [x] Registrar evidencias, file list e resultado de QA
