# Brand Audit: BrandOps

## Snapshot
- **Estado atual:** premium-operacional parcial, com maior maturidade visual na `MEDIA LIBRARY`
- **Forca principal:** densidade funcional, hierarquia clara e shell de produto consistente
- **Gap principal:** a marca do produto e a marca dos clientes ainda nao governavam a interface de ponta a ponta

## Preserve
- arquitetura por projeto
- linguagem de dashboard operacional
- densidade e composicao da media library
- design-system baseado em tokens semanticos

## Evolve
- logo e wordmark do produto
- herois, sidebar, cards e CTA
- traducao da identidade do cliente para a shell
- qualidade premium de dark/light mode

## Remove
- atmosfera visual genérica de SaaS
- violeta padrao sem intencao de marca
- qualquer separacao entre `BRAND CORE` e shell real do projeto

## Direction
- **BrandOps:** luxo estrategico editorial-tech
- **Cliente:** shell completa tematizada por projeto
- **Produto:** precisa parecer uma plataforma executiva high ticket, nao uma ferramenta utilitaria comum
