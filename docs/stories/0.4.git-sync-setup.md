## Risks

- **rclone OAuth Complexity:** Google Drive authentication via rclone requires OAuth flow and token management. Mitigation: Complete rclone-config.md with step-by-step OAuth setup instructions, test credentials before production scheduling
- **Sync Conflict Scenarios:** Bi-directional sync with local-wins strategy may cause data loss if both local and remote change simultaneously. Mitigation: Document conflict resolution behavior clearly, implement metadata tracking in .sync-metadata/ folder to prevent accidental overwrites
- **Windows Task Scheduler Requirements:** Scheduler task must run with admin privileges and proper network connectivity. Mitigation: Document privilege requirements, include failure retry logic (3 attempts), log all sync operations to Windows Event Log
- **File Permission Issues:** Windows NTFS permissions on local storage and Google Drive permissions may conflict. Mitigation: Document required permissions for E:\BRAND-OPS-STORAGE\ folder, test rclone access before deploying scheduled task
