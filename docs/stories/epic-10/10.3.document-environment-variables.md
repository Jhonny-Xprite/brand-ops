## `.env.example` Template

```bash
# Database Configuration
DATABASE_URL="file:./prisma/dev.db"
# SQLite database file path for local development

# Environment
NODE_ENV="development"
# development | production
# Controls logging level and optimization

# File Upload Configuration
MAX_FILE_SIZE_MB="50"
# Maximum file size in megabytes
# Used in Story 10.2

ALLOWED_FILE_TYPES="jpg,jpeg,png,gif,mp4,mov,webm,psd,ai"
# Comma-separated list of allowed file types
# Used in Story 10.2

# Logging
LOG_LEVEL="debug"
# Optional: debug | info | warn | error
# Default: info
```
