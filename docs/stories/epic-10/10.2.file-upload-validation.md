## Story

**As a** security-conscious developer,  
**I want** file uploads to be validated for type and size,  
**so that** the application rejects invalid files before processing.

## Dev Notes

- File validation should happen in API route handler
- Use MIME type whitelist, not blacklist (security best practice)
- Check file extension AND MIME type (some can be spoofed)
- Use the `busboy` library already in project for stream validation
- Error messages should be user-friendly but not reveal security details
