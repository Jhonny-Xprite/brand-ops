[Source: architecture/tech-stack.md§Backend Stack]

- SQLite: Latest (bundled with better-sqlite3)
- TypeScript: strict mode enabled

[Source: architecture/coding-standards.md§TypeScript Standards]
- No `any` types
- Strict mode required
- All Prisma models must have TypeScript interfaces

### Project Structure

```

├── prisma/
│   ├── schema.prisma        # Data model definitions
│   └── migrations/          # Schema version history
│       └── {timestamp}_init/

├── data/
│   └── brand-ops.db         # SQLite database (gitignored)
├── .env.local               # Database URL (DO NOT COMMIT)
└── src/
    └── lib/
        ├── prisma.ts        # Singleton Prisma Client instance
        └── __tests__/
            └── prisma.test.ts  # CRUD tests
```
