## Story
**Como** operador navegando dentro de um projeto,  
**Eu quero** que a `MEDIA LIBRARY` use a mesma composicao operacional da referencia aprovada e que a sidebar exponha um item explicito de `CONFIG`,  
**Para que** a navegacao do projeto fique coerente e a biblioteca de midia deixe de parecer quebrada ou espremida.

## Acceptance Criteria
1. `/projeto/[id]/media` deixa de usar um shell aninhado e passa a renderizar a biblioteca em um layout limpo de workspace + painel lateral
2. A `MEDIA LIBRARY` aproxima sua composicao da referencia aprovada: cabecalho de biblioteca, filtros em linha, barra de acoes, grade/lista e painel lateral
3. Os cards de assets e o estado vazio ficam visualmente consistentes com a biblioteca de referencia
4. A sidebar do projeto passa a exibir um item explicito de `CONFIG`
5. `lint`, `typecheck`, `build` e `test` passam ao final

## Tasks

- [x] Reorganizar header, filtros, barra de acoes e resultados para aproximar a `MEDIA LIBRARY` da referencia aprovada
- [x] Simplificar os cards da `FileList` para o novo baseline de biblioteca
- [x] Expor o item `CONFIG` na sidebar do projeto
