# Story 0.4: Página de Configurações da Marca (Design System)

## Status
Ready

## Story
**Como** um curador de marca,
**Eu quero** definir as cores, fontes e logotipos base de um projeto,
**Para que** o sistema use esses elementos em toda a interface e na organização de ativos.

## Acceptance Criteria
1. **Inputs de Identidade:** Campos para definir HEX das cores Primária e Secundária.
2. **Preview de Cores:** Mostrar uma pequena amostra visual ao lado do código HEX.
3. **Seleção de Tipografia:** Dropdown para escolher fontes de Título e Corpo (Sora/Inter padrão).
4. **Gestão de Logos:** Área de upload para "Logo Principal" e "Símbolo/Ícone".
5. **Persistência:** Salvar essas configurações no SQLite associadas ao `project_id`.
6. **Interface:** Seguir o padrão de Dark Mode Premium (superfícies escuras, destaques em violeta/ouro).

## Tasks
- [ ] Criar página `/projeto/[id]/config`
- [ ] Implementar formulário de Branding com React Hook Form
- [ ] Criar lógica de upload e armazenamento de logos de marca
- [ ] Integrar persistência no Prisma (modelo `ProjectConfig`)
- [ ] Validar a aplicação visual dos tokens salvos em um painel de "Preview"
