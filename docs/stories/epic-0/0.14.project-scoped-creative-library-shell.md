## Story
**Como** operador trabalhando dentro de um projeto,  
**Eu quero** uma Creative Library exclusiva daquele contexto e um shell lateral no mesmo idioma visual da biblioteca criativa,  
**Para que** eu navegue entre as areas do projeto sem perder o contexto nem depender de uma barra superior separada.

## Acceptance Criteria
1. `/projeto/[id]/media` entrega a Creative Library do projeto no mesmo estilo operacional da `/creative-library`
2. Upload e leitura de arquivos em `media` respeitam o escopo do projeto via tags `project-{id}` e `scope:media`
3. O shell do projeto deixa de usar menu superior e passa a usar navbar lateral estilo Creative Library
4. Todos os menus do projeto ficam na sidebar lateral
5. Metadata e refresh de versionamento preservam o contexto do projeto ao recarregar a lista
6. `lint`, `typecheck`, `build` e `test` passam ao final

## Tasks
