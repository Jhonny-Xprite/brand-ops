## Story
**Como** gestor de multiplos projetos,  
**Eu quero** uma home global para listar, buscar e criar projetos rapidamente,  
**Para que** eu entre no contexto correto sem friccao e com visibilidade do estado geral do workspace.

## Acceptance Criteria
1. A home global oferece visualizacao em grid e lista com persistencia local da preferencia
2. Cada projeto mostra nome, ativo visual basico e contagem resumida de assets relacionados
3. O botao de novo projeto abre modal funcional para criar o projeto e opcionalmente enviar logo inicial
4. A busca filtra os projetos localmente em tempo real
5. O rodape mostra o estado global de sync/offline de forma coerente com o sistema
6. `lint`, `typecheck`, `build` e `test` passam no fechamento da story

## Tasks

- [x] Implementar componentes de home global, busca, toggle e modal

- [x] Conectar o rodape de sync ao estado global do workspace
