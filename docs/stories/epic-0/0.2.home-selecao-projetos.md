# Story 0.2: Tela de Seleção de Projetos (Home Global)

## Status
Ready

## Story
**Como** um gestor de múltiplas marcas,
**Eu quero** uma tela centralizada para visualizar todos os meus projetos,
**Para que** eu possa entrar no contexto específico de um cliente com rapidez e organização.

## Acceptance Criteria
1. **Visualização Dupla:** Toggle para alternar entre "Grade" (Cards grandes com logos) e "Lista" (Tabela densa).
2. **Cards de Projeto:** Cada card deve exibir Logo, Nome do Projeto e total de ativos.
3. **Criação de Projeto:** Botão "Novo Projeto" que abre modal para definir Nome e fazer upload do Logo inicial.
4. **Busca Rápida:** Campo de busca que filtra a lista de projetos localmente por nome.
5. **Feedback de Sincronização:** Rodapé da página deve mostrar o status global de sync do repositório.
