## Story
**Como** estrategista de marca operando dentro do projeto,  
**Eu quero** uma area de estrategia funcional e uma entrada de media coerente com a Creative Library compartilhada,  
**Para que** eu consiga acessar materia-prima estrategica e arquivos de media sem duplicar experiencias nem quebrar a arquitetura do produto.

## Acceptance Criteria
1. `/projeto/[id]/strategy` deixa de ser placeholder e entrega superficie funcional
2. `/projeto/[id]/media` evolui apenas como superficie de integracao com a Creative Library compartilhada
3. Upload, metadata editor, versionamento e browse de media nao sao duplicados
4. O contrato de dados de projeto para strategy/media fica centralizado e sem gambiarra
5. Navegacao, layout e linguagem visual seguem o shell compartilhado
6. `lint`, `typecheck`, `build` e `test` passam
